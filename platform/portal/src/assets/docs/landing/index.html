Quick Start Guide

Welcome to the Platform documentation. Here is how you can bootstrap your environment in minutes.

curl -sL https://platform.io/install.sh | bash
platform bootstrap --prod

Once the cluster is up, grab your default credentials from the generated secrets file and head to the dashboard.

SDK Ecosystem

Plug in any stack. Get insights everywhere.

Node.js @platform/sdk-node

Express middleware, structured logger, heartbeat manager.

npm install @platform/sdk-node

React @platform/sdk-react

Context provider, Axios interceptors, bug reporter widget.

npm install @platform/sdk-react

Python @platform/sdk-python

Django/Flask compatible. Decorator-based tracking.

pip install platform-sdk-python

Angular @platform/sdk-angular

HTTP interceptor module, injectable CapsService.

npm install @platform/sdk-angular

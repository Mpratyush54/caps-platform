// Initialize Platform

await plat.init({
  projectName: 'my-backend',
  platformUrl: 'https://api.platform.dev'
});

// Zero-overhead tracing for Express
app.use(plat.expressMiddleware());

// Forward all console logs to Loki
plat.captureConsole();

How it Works

From local development to production Kubernetes clusters in three simple steps.

1.

Install Platform

Run our helm chart or bootstrap script on your cluster. It provisions k3s, ArgoCD, MinIO, and Prometheus automatically.

2.

Init the SDK

Drop our SDK into your Node.js, Python, or React app. It instantly starts tracing requests, catching bugs, and streaming logs.

3.

Scale & Monitor

Open the Console to see live metrics, manage databases, roll back deployments, and view secret audit logs in real-time.

Kubernetes (k3s)

ArgoCD

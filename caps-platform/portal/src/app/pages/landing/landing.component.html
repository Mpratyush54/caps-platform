Overview

An open-source, self-hosted PaaS that provides a unified control plane for deploying, managing, and monitoring your applications. It runs on Kubernetes (k3s) and integrates with GitLab/GitHub for CI/CD, ClickUp for project tracking, and Infisical for secrets management.

Webhooks

      
The platform can receive webhooks from GitHub and GitLab to automatically trigger deployments.


      
GitHub

      
    
        
  • Endpoint: POST /api/webhooks/github
    • 
        
  • Signature: Verified via X-Hub-Signature-256 (HMAC SHA256)
    • 
        
  • Events: push (preview on feature, staging on main), pull_request (preview on open/sync, terminate on close)
    • 
      


      
GitLab

      
    
        
  • Endpoint: POST /api/webhooks/gitlab
    • 
        
  • Token: Verified via X-Gitlab-Token
    • 
        
  • Events: Push Hook (same logic as GitHub), Merge Request Hook (preview on open, terminate on close/merge)
    •
